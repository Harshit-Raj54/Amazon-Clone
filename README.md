# Amazon UI Clone

This is a front-end **Amazon homepage UI clone** developed using **HTML, CSS, and JavaScript** as a learning project.

⚠️ This project is created **only for educational purposes** and does not represent the official Amazon website.

---

## Features
- Amazon-style navigation bar and search section
- Product cards and category layout
- Hover effects and responsive sections
- Scroll-to-top functionality
- Basic JavaScript interactivity

---

## Technologies Used
- HTML5
- CSS3 (Flexbox, animations, hover effects)
- JavaScript (DOM manipulation & event handling)

---

## Project Purpose
This project was built to:
- Improve front-end layout skills
- Practice real-world UI cloning
- Gain hands-on experience with CSS positioning and DOM events
